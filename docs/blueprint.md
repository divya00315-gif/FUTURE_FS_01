# **App Name**: Portfolio Pro

## Core Features:

- Responsive Public Portfolio: Display a responsive portfolio with sections for home, about, skills, projects, and contact.
- Project Display: Fetch project data from the MongoDB database and display it.
- Contact Form Submission: Allow users to submit messages, which are then stored in MongoDB and shown as a success message.
- Admin Login: Secure admin login with username and password stored safely, providing access to project management and contact message viewing.
- Project Management: Admin panel to add, edit, and delete projects.
- Contact Message Viewer: View contact messages in the admin panel.
- Resume Generation: Generative AI tool that intelligently pulls your information and project descriptions from the database and contact form submissions, in order to compose a compelling summary of qualifications.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and professionalism.
- Background color: Light Gray (#EEEEEE), providing a clean, modern backdrop.
- Accent color: Teal (#009688) to highlight important sections and call-to-action elements.
- Body font: 'PT Sans', a sans-serif font for readability and a modern feel.
- Headline font: 'Space Grotesk', a sans-serif font, lending a technological edge
- Use a set of professional icons to visually represent skills and technology.  For the project, try to source project specific image assets
- Subtle animations when scrolling through sections to provide a smooth user experience.
